Exercice 14.

Encoder les messages suivants avec le système RSA dont les paramètres sont précisés avec le message.

(a)

Mon enseignant est le meilleur avec paramètres \(pq=2077, e=15841\)

(b)

Je vais faire tous les devoirs avec paramètres \(pq=18871, e=3261\)

(c)

Un jour je serai le meilleur dresseur avec paramètres \(pq=45113,e=5555\)

(d)

Qui a-t-il de particulier avec ces messages codés? Est-ce qu'il y a une lettre qui est facile à décoder? En utilisant des blocs de lettres comme dans le vrai RSA, ceci ne se produit pas.
